# Neuroid Angular 16 Demo

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.2.1.

## Step 1 

Run `ng build`

## Step 2

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`
